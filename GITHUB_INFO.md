## 🚀 快速开始

### 克隆项目

```bash
git clone https://github.com/corlin/fineagent.git
cd fineagent
```

### 安装依赖

```bash
uv sync
```

### 运行示例

```bash
# 运行第一个示例
uv run python src/example_01_basic_chat.py

# 交互式运行
uv run python src/run_all_examples.py
```

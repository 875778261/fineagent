这是一个完整的 Langchain 1.1.0 学习项目，包含 15 个循序渐进的示例
（基础 10 个 + 高级 5 个），基于统一的"智能旅行规划"场景，全面展示
Langchain 从入门到生产级应用的核心能力。

🌟 核心特色:
  • 统一场景设计，便于理解和对比
  • 循序渐进的学习路径
  • 生产级高级示例
  • 完整的文档体系（11个文档）
  • 所有代码可直接运行

🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 安装依赖
   $ uv sync

2. 运行基础示例
   $ uv run python src/example_01_basic_chat.py

3. 运行高级示例
   $ uv run python src/example_11_multi_agent.py

4. 交互式体验
   $ uv run python src/run_all_examples.py
